# 인터넷과 웹
1) 인터넷과 웹
   * 웹: 링크방식을 통해..
2) 웹개요
   * 웹페이지(HTML)를 해석하는 SW필요: 웹브라우저
   * 웹페이지를 저장하고 공유하기 위한 공유저장소: 웹서버(Nginx)
   * 링크를 통한 정보의 무한 연결성 제공
   * 컴퓨터와 컴퓨터간 통신은 IP로 통신
   * 웹브라우저 컴퓨터와 웹서버 컴퓨터간 통신하려면 웹서버의 IP를 알아야함
